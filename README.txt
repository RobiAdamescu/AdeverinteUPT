Adresa repository: https://github.com/RobiAdamescu/AdeverinteUPT
1.Se clonează repository-ul folosind comanda: git clone https://github.com/RobiAdamescu/AdeverinteUPT

2.Se instalează editorul de cod VS Code și Angular.
Directorul AngularAuthUI reprezintă interfața frontend.
Pentru instalarea pachetelor se folosește npm install în terminalul din VS Code. După instalarea pachetelor aplicația poate rula.
Se folosește comanda ng serve -o pentru rularea aplicației frontend.

3.Se instalează editorul de cod Visual Studio 2022 pentru serverul backend.
Se apasă click dreapta pe numele proiectului în visual studio 2022 și se selecteaza manage NuGet packages.
Se instalează urmatoarele pachete si urmatoarele versiuni:
DocumentFormat.OpenXml 3.0.2
DocX 3.0.0
Mammoth 1.4.0
Microsoft.AspNetCore.Authentication.JwtBearer 6.0.29
Microsoft.EntityFrameworkCore 6.0.29
Microsoft.EntityFrameworkCore.Tools 6.0.29
Npgsql.EntityFrameworkCore.PostgreSQL 6.0.29
Swashbuckle.AspNetCore 6.4.0

După instalarea pachetelor, aplicația poate rula, apasând tasta f5.
